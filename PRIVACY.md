# Privacy

Sokoban Web is a static browser game. It has no application server, user accounts, advertising, or analytics.

The game stores the following data in your browser's `localStorage`:

- Theme preference under `sokoban-theme`
- Saved game state under keys beginning with `sokoban:v1:`

The selected level is also placed in the page URL so it can be bookmarked or shared. This data is not transmitted to the project maintainer by the application. You can remove stored data at any time through your browser's site-data controls.

Links to GitHub and donation providers leave the application and are governed by those services' privacy policies. GitHub Pages may process standard request metadata as described in GitHub's own privacy statement.

Privacy questions can be submitted through the project's [issue tracker](https://github.com/doctorlai/sokoban-web/issues), without including sensitive personal information.
