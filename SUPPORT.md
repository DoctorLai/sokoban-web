# Support

## Getting help

Search the [existing issues](https://github.com/doctorlai/sokoban-web/issues) first. If the problem has not been reported, open a new issue using the most relevant template.

Include the level ID, browser and operating system, steps to reproduce, expected result, and screenshots or console errors when relevant. Please do not post account details, secrets, or security vulnerabilities in public issues.

For suspected security issues, follow [SECURITY.md](SECURITY.md).
