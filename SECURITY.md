# Security Policy

## Supported version

Sokoban Web is deployed continuously from the `main` branch. Only the latest deployed version is supported with security fixes.

## Reporting a vulnerability

Please do not disclose suspected vulnerabilities in a public issue. Use [GitHub's private vulnerability reporting](https://github.com/doctorlai/sokoban-web/security/advisories/new) and include:

- A description of the issue and its impact
- Reproduction steps or a minimal proof of concept
- Affected browsers, versions, or URLs
- Any suggested mitigation

You should receive an initial response within seven days. Confirmed issues will be addressed according to their severity and disclosed after a fix is available.
