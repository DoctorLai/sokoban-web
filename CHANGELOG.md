## [2026-07-16]

### Added

- Unified formatting, type checking, testing, coverage, and build scripts
- Pull request coverage reports with an 80% minimum for every metric
- Repository contribution, security, support, privacy, and dependency-update policies
- Date and commit version information in the deployed interface

### Changed

- Updated development dependencies to versions without known npm audit findings
- Standardized CI and GitHub Pages deployment on Node.js 22
- Pinned workflow actions to immutable commits
- Kept fork pull request coverage checks read-only and removed duplicate type checking

### Fixed

- Made solution playback start from the board state that was actually solved
